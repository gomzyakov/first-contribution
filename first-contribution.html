Tutorial

Your first contribution

Follow these steps to create your first pull request. You can safely experiment here.

Step 1

Click the Fork button on the repository page to create your own copy under your GitHub account.

Step 2

Clone your fork to your local machine. Replace your-username with your actual GitHub username.

git clone https://github.com/your-username/first-contribution.git
cd first-contribution
Step 3

Add an upstream remote to keep your fork up to date with the original repository.

git remote add upstream https://github.com/your-org/first-contribution.git
git remote -v
Step 4

Create and switch to a new branch for your changes. Use a descriptive branch name.

git checkout -b add-your-name
Step 5

Edit the CONTRIBUTORS.md file and add your name. Save the file when you are done.

Step 6

Stage your changes and commit with a clear message.

git add CONTRIBUTORS.md
git commit -m "Add your-name to contributors list"
Step 7

Push your branch to your fork on GitHub.

git push origin add-your-name
Step 8

Go to your fork on GitHub and click Compare & pull request. Describe your change and submit the PR.

Tip: If you get stuck, search for the exact error message or ask for help in project discussions.
